# Recruiter Notes

## Project Pitch

Stealth AA Recovery Vault is a smart account research implementation focused on private social recovery and account abstraction. It avoids standard token/NFT patterns and instead demonstrates protocol-level Solidity engineering.

## Skills Demonstrated

- Solidity `^0.8.24`
- Account abstraction architecture
- EIP-1271 smart contract signature validation
- Commit-reveal protocol design
- Social recovery security modeling
- Session key authorization
- Sponsored gas budget accounting
- Hardhat tests
- Gas-conscious coding patterns

## Interview Talking Points

- Why storing guardian addresses openly creates metadata risk.
- Why recovery needs both a threshold and a time delay.
- Why EIP-712 is better than raw signed messages for production.
- How ERC-4337 validation differs from normal wallet calls.
- How session keys can improve UX while creating permission-scope risk.
